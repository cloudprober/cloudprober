Cloudprober

Monitoring software to detect failures before your customers do.

Get Started

Open-source. GitHub

Fast native checks ⚡️

Cloudprober comes with efficient, highly scalable, built-in probes for common protocols: HTTP (REST APIs), PING, DNS, gRPC, TCP, UDP.

Trusted and used by multiple large organizations

Google, Apple, DoorDash, Cloudflare, Yahoo! JAPAN, Goldman Sachs, Robinhood, Hostinger, and more.

Easy integration

Out of the box, config based, integration with many popular monitoring systems: Prometheus, Grafana, DataDog, PostgreSQL, AWS Cloudwatch, Stackdriver, and more.

Automated targets discovery

Reliable, automated targets discovery for Kubernetes and GCP resources.

Deployment friendly

Written entirely in Go. Available as standalone static binary or through a docker image. Multiple ways to configure, including environment variables.

Run arbitrary code as a probe

Run any binary or script as a probe, and have probe results appear in standard metrics.
